# Creepypasta Store

Данный проект представляет собой полноценное Fullstack-приложение для управления каталогом товаров. В рамках проекта реализован современный интерфейс магазина с возможностью управления ассортиментом в реальном времени, где можно просматривать, фильтровать и управлять каталогом крипипаста-персонажей.

## Структура проекта

* `/backend` — сервер на Express (порт 3001).
* `/frontend` — клиентская часть на React (порт 3000).
    * `src/pages/ProductsPage` — основная страница магазина.
    * `public/img` — хранилище локальных ассетов и фонов.

## Окружение и API

Для корректной работы магазина необходимо запустить обе части приложения (Frontend и Backend). Проект использует следующую сетевую конфигурацию:

* **Frontend**: [http://localhost:3000](http://localhost:3000) — основной интерфейс витрины.
* **Backend API**: [http://localhost:3001](http://localhost:3001) — базовый URL сервера.
* **Swagger UI**: [http://localhost:3001/api-docs](http://localhost:3001/api-docs) — интерактивная документация для тестирования сущностей.

### Спецификация эндпоинтов

Все операции с товарами выполняются через префикс `/api/products`.

| Метод | Эндпоинт | Описание |
| :--- | :--- | :--- |
| **GET** | `/api/products` | Получить полный список активных сущностей магазина. |
| **GET** | `/api/products/:id` | Получить детальное описание конкретной сущности по её ID. |
| **POST** | `/api/products` | Вызвать (создать) новую сущность в базе данных. |
| **PATCH** | `/api/products/:id` | Изменить характеристики или уровень опасности существующей сущности. |
| **DELETE** | `/api/products/:id` | Изгнать (удалить) сущность из текущего измерения. |
